## 自己PR
私は、デザインエンジニアとしての多岐にわたるスキルと経験を活かし、UIデザインからフロントエンド開発、Webアクセシビリティ改善まで、幅広い領域で成果を出してきました。
特に、Ubie株式会社ではデザインシステムの構築に注力し、デザインの均質化とフロントエンドの生産性向上を推進しました。デザイントークンの標準化や、FigmaとGitHub Actionsの連携によるワークフローの自動化を実現し、効率的なデザインプロセスを構築しました。

また、Webアクセシビリティの改善プロジェクトでは、障害者インタビューやアクセシビリティ試験を通じてユーザー体験の向上に貢献しました。これにより、プロダクトが「情報アクセシビリティ好事例2023」に選出されるなど、具体的な成果を上げています。

さらに、複数のプロジェクトでの経験から、React、Next.js、GraphQLなどのモダンなフロントエンド技術に精通しています。
特に、Next.jsを用いた社内管理画面の開発や、個人向け医療情報の記録機能の開発では、UIデザインとエンジニアリングの両方の視点を活かし、ユーザーにとって使いやすく機能的なインターフェースを提供しました。

これまでの職務経験で培ったデザインとエンジニアリングのスキル、そしてユーザー視点での開発力を武器に、より多くのユーザーにとって価値のあるプロダクトの提供を目指していきます。
